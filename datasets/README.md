# Golden Test Datasets

Committed fixtures for reproducible testing. Both R and Python read these identical files.

| File | Task | n | Target | Use |
|------|------|---|--------|-----|
| iris_binary.csv | binary | 50 | species (setosa vs versicolor) | Binary classification |
| iris_multiclass.csv | multiclass | 60 | species (3 classes) | Multiclass classification |
| mtcars_regression.csv | regression | 32 | mpg | Regression |
| text_sentiment.csv | text/binary | 35 | sentiment (pos/neg/neutral) | Text + TF-IDF |

## expected/

Golden outputs generated by Python (reference implementation) with `seed=42`.
R parity tests compare against these within stated tolerance (1e-6 tabular).

## Rules

- NEVER modify existing datasets (add new ones instead)
- All datasets must have a header row
- No missing values in golden datasets (test missing values separately)
- Text datasets: no special characters that break CSV parsing
